Skiper UI

freemium

Premium, uncommon components you don't see everywhere. The dynamic island and image cursor-trail, dropped straight into your landing page.

Visit skiper-ui.com

02 / 06

Vengeance UI

free · OSS

Free, open source, and good-looking. Glass docks, spotlight navbars, all with one clean single-line install.

Visit vengenceui.com
screenshot → assets/02-vengeance.png
via shadcn MCP
03 / 06

Animmaster Lib

from ~$3

Over three hundred animations pulled from award-winning websites, for the price of a coffee. A little old-school, you copy and paste it in yourself.

Visit animmasterlib.dev
screenshot → assets/03-animmaster.png
no MCP
04 / 06

Fancy Components

free · OSS

Pure fun. Real physics, so letters actually fall and crash into each other, plus an image trail that chases your cursor around the page.

Visit fancycomponents.dev
screenshot → assets/04-fancy.png
no MCP
05 / 06

React Bits

free · OSS

This one feels like the future. Full-screen WebGL shaders, liquid chrome, iridescence, aurora, and it ships its own MCP. Tell your editor what you want and it drops the shader right in.

Visit reactbits.dev
screenshot → assets/05-reactbits.png
its own MCP

SmoothUI

The cleanest of the bunch to play with. Dynamic island, Siri orb, number-flow. Same Skiper energy, buttery to demo.

Visit smoothui.dev
